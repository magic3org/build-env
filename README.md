Scripts and configration to build Magic3 environment
====

Easily build Magic3 environment by Vagrant

## Requirement

- VirtualBox

https://www.virtualbox.org/wiki/Downloads

- Vagrant

https://www.vagrantup.com/downloads.html

## Usage

1. make work directory

    $ mkdir dirname

1. copy Vagrantfile

    $ cd dirname
    $ cp Vagrantfile ./

1. vagrant up

    $ vagrant up
